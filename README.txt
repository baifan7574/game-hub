
# GoGameFun 使用说明

这是一个基础版小游戏网站框架，包含：
- 首页入口；
- 两个游戏页面（Game1为可玩游戏）；
- 联系方式页面；
- 正确的图片路径结构。

## 如何使用
1. 解压整个文件夹。
2. 打开 index.html 测试是否能运行。
3. 将整个文件夹上传至你的 GitHub 仓库。
4. 提交后启用 GitHub Pages。

## 游戏说明
- Game1 是一个可玩的打飞机小游戏（HTML + JS）。
- Game2 是占位页面，可替换为其他 HTML 游戏。

## 图片
将你上传的游戏图命名为 game2.png，放入 images 文件夹中即可显示。
